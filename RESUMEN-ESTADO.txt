QUÉ HACER AHORA
===========================================

PASO 1: Instalar XAMPP
-----------------------
1. Ve a: https://www.apachefriends.org/download.html
2. Descarga XAMPP para Windows (versión con PHP 8.x)
3. Instala en C:\xampp\ (ubicación por defecto)
4. Abre el Panel de Control de XAMPP
5. Haz clic en "Start" para Apache
6. Haz clic en "Start" para MySQL

PASO 2: Crear Base de Datos
----------------------------
1. Abre tu navegador: http://localhost/phpmyadmin
2. Haz clic en "Nueva" (New) en el menú lateral
3. Nombre: delprado_db
4. Cotejamiento: utf8mb4_unicode_ci
5. Haz clic en "Crear"
6. Selecciona la base de datos delprado_db
7. Ve a la pestaña "Importar" (Import)
8. Selecciona el archivo database.sql de este proyecto
9. Haz clic en "Continuar" (Go)

PASO 3: Iniciar el Servidor
----------------------------
Abre PowerShell en esta carpeta y ejecuta:

    C:\xampp\php\php.exe -S localhost:8000

O si agregaste PHP al PATH:

    php -S localhost:8000

PASO 4: Acceder al Panel
-------------------------
Abre tu navegador en:

    http://localhost:8000/admin/login.php

    Usuario: admin
    Contraseña: admin123
